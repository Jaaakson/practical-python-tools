## ğŸš€ How to Use

Follow these steps to get everything running on your machine.

### 1. Clone the Repository
Open your terminal and run the following commands to download the code and enter the folder:

```bash
git clone https://github.com/Jaaakson/practical-python-tools
cd practical-python-tools
```

### 2. Install Dependencies
This project uses a requirements.txt file to manage all necessary libraries (like qrcode and Pillow) in one go. Run this command:
```bash
pip install -r requirements.txt
```
### 3. Run an Application
Navigate to the specific tool you want to use and run the main.py file.

Example: Running the QR Generator
```bash
cd qr-generator
python main.py
```
